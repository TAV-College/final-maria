RecipeBox is an Expo mobile application built with React Native and TypeScript.  
The app retrieves recipe data from an external API and displays related information such as categories, meals, and ingredients.

---

## Features

- Built with **Expo** and **TypeScript**
- Fetches data from the internet using **TheMealDB API**
- Uses **React Context** to store favorite meals
- Navigation between multiple screens
- Displays related data objects (categories ‚Üí meals ‚Üí meal details ‚Üí ingredients)

---

## Screens (Pages)

- Categories
- Meals by Category
- Meal Details
- Favorites

---

## Components

- CategoryCard
- MealCard
- IngredientRow
- Loading
- ErrorState
- FavoriteButton
